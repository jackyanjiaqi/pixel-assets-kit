# pixel-assets-kit 像素风格包(大屏游戏素材包)
Pixel Cardboard Split-Screen App Developing Took Kits
使用网格矩阵 绘制文字，作为像素美术风格的资源包，跟随 Pixel Cardboard Launcher 一起发布

# 演示DEMO

![演示](/pic/pixel_large.png)

# 功能概述

## 使用字符串表示'像素'

## 根据'像素'字符串绘制文字或图形

## 不使用矢量直接绘制文字或图形，内嵌在游戏场景中，具有风格化特点
